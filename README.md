# 🪕 Balalaika Hero

**Balalaika Hero** — это динамичная браузерная ритм-игра в уникальном сеттинге цифрового фольклора. Проект был разработан в рамках Хакатона всего за 24 часа, пройдя путь от идеи до рабочего продукта.

[**🎮 ИГРАТЬ В БРАУЗЕРЕ**](https://bocoder17.github.io/balalaika-hero/)

---

## 🚀 Что реализовано

*   **Dynamic Difficulty**: Три уровня сложности (Легкая, Средняя, Сложная). С ростом уровня увеличивается скорость полета матрешек, их плотность, а также количество опасных бомб и полезных бонусов.
*   **Visual Feedback**: Система мгновенной обратной связи — анимации танца медведя при попадании и визуальные индикаторы («МИМО») при промахах.
*   **Input System**: Управление — поддержка клавиатуры (`A`, `S`, `D`).
*   **Retro Aesthetics**: Авторский пиксель-арт, традиционные орнаменты и шрифт *EpilepsySans* создают атмосферу «Cyber-Folk».

## 📦 Быстрый старт

Для запуска локально установите `Node.js`.
```bash
# 1. Клонирование репозитория
git clone 
cd balalaika-hero

# 2. Установка зависимостей
npm install

# 3. Запуск dev-сервера
npm run dev
```
